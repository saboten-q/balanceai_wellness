# Assets フォルダ

このフォルダには、以下のアセットファイルを配置してください：

## 必要なファイル

1. **icon.png** (1024x1024 px)
   - アプリのアイコン画像

2. **splash.png** (1242x2436 px または同様のアスペクト比)
   - アプリ起動時のスプラッシュ画面

3. **adaptive-icon.png** (1024x1024 px)
   - Android用のAdaptive Icon

4. **favicon.png** (48x48 px)
   - Web版用のファビコン

## 簡易的な作成方法

オンラインツールを使用して簡単に作成できます：
- [Canva](https://www.canva.com/)
- [Figma](https://www.figma.com/)

または、Expoが提供するデフォルトアイコンを使用することもできます。
